# Tomer Dayboard

גרסה עם מאגר קלוריות אוטומטי ממוצע:
- כתוב מאכל + כמות בגרמים, לדוגמה: `שווארמה 200 גרם` או `פיצה 150 גרם`.
- האפליקציה משלימה אוטומטית קלוריות, חלבון ושומן לפי ממוצע ל-100 גרם.
- מאכל שלא מוכר במאגר יעבור להזנה ידנית.
- ספר מוצרים אישי: אפשר לשמור מוצר עם ערכים וכמות, ובהמשך הוא יחושב אוטומטית לכל כמות שתכתוב.
- הערכים הם הערכות ממוצעות; אם יש אריזה/מסעדה עם ערכים מדויקים, עדיף לשמור אותם בספר מוצרים.

## הרצה
```bash
npm install
npm run dev
```
