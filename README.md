# Tomer Daily Planner

גרסה זו מתקנת את השורה התחתונה באייפון:
- הניווט התחתון נשאר fixed ולא נפתח/נדחף מוזר.
- נוספה תמיכה טובה יותר ב-safe-area של iPhone.
- הטאבים מוצגים כ-grid יציב עם 5 עמודות.
- נשמרו כל הפיצ׳רים הקיימים.

הרצה מקומית:
```bash
npm install
npm run dev
```

בנייה:
```bash
npm run build
```
