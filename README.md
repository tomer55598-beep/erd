# Tomer DayBoard

אפליקציית יום אישית: משימות, תזונה עם חישוב קלוריות אוטומטי לפי גרמים, מים, משקל והיסטוריה.

## הרצה מקומית
```bash
npm install
npm run dev
```

## Build ל־Vercel
```bash
npm run build
```

הגרסה הזו כוללת שיפור UI: כרטיס פתיחה, כרטיסים מודרניים, ניווט תחתון משופר, התאמות לאייפון ושיפור נראות כללית.
